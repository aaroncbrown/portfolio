Restaurant Poll

I just started working on this so it is not on GitHub, but it will be a website that will help groups decide where they want to eat. Using the site someone can choose restaurants in the area and send out a poll where the members of the group can vote where they want to go. The winning restaurant will be sent out to everyone in the group.
I am separating this project into an Angular / TypeScript front end and an ASP.Net Core / C# back end API that utilizes the Google Maps API. If everything goes well I would like to turn this into a mobile app.
I am creating this project using the following technologies:

  • Angular
  • TypeScript
  • Bootstrap
  • Font-Awesome
  • ASP.Net Core
  • C#
  • Entity Framework Core
  • SQL Server
  • Google Maps API
